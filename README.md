# Breast Cancer Classification using K-Nearest Neighbors

This project demonstrates how to build a K-Nearest Neighbors (KNN) classifier using the Breast Cancer Wisconsin dataset from scikit-learn.

## Features

- Exploratory data analysis
- 3D feature visualization
- PCA projection
- KNN classifier
- Decision boundary visualization
- Model evaluation

## Technologies

- Python
- pandas
- NumPy
- scikit-learn
- matplotlib

## Results

Model Accuracy: 94%

## Dataset

Breast Cancer Wisconsin Dataset
Available in scikit-learn.
